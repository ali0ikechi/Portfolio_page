This was the first project that I have completed from Udacity Front End Developer Nanodegree. The personal blog was finished using pure HTML&CSS to format the style with Flexbox and Grid.

Weather App

This is a single-page weather applicaiton that I have completed for SheCodes React.js course.The project used React.js and a third-party API OpenWeather.

The page is loaded with the location of Berlin showing degrees in celcius, but you can search any cities accross the world and change the metric.
The design minimized the color themes to focus on the information to be stood out.
